# OptoCL
Closed-loop control of sinusoidal optogenetic stimulation

## Running
See:
- OptoCL_run.m (MATLAB)
- OptoCL_run.s2s (Spike2)

## Requirements
Tested with MATLAB R2021a (Mathworks) and Spike2 7v.20 (CED).
